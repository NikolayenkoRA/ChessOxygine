Проект шахмат на языке C++ с использованием Oxygine Engine (https://github.com/oxygine), сборка осуществляется под Linux
На данный момент реализована логика ходов всех фигур, поедание фигур, проверка очередности хода и добавлен UI с сообщениями (невозможность хода, очередность хода)

Для работы склонировать себе oxygine-framework (https://github.com/oxygine/oxygine-framework) рядом с этим репозиторием, а также подгрузить SDL2 библиотеки (требует движок)
